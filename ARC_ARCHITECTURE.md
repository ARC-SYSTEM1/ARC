# ARC System Architecture
## Arena. Rhythm. Culture.

### Overview
ARC is a cloud-based operating intelligence platform designed to monitor, orchestrate, and optimize physical environments in real time.

### Core System Flow
Signals → State → Decision → Action → Environment → ROI

### Core Components

#### 1. Input Layer
- Sensors (Aqara, IoT devices)
- Operator inputs
- Revenue and guest data
- Voice triggers via Voice Monkey

#### 2. State Engine
- Maintains real-time system context
- Tracks presence, activity, and energy

#### 3. Decision Engine
- Deterministic automation logic
- Applies rule-based orchestration

#### 4. Environment Controller
- Controls lighting, alerts, and integrations
- Interfaces with IoT devices

#### 5. Intelligence Layer
- KPI tracking and ROI analytics
- Night replay and performance insights

#### 6. Interface Layer
- ARC Operator Dashboard
- API endpoints
- Voice integrations

#### 7. Infrastructure Layer
- FastAPI backend
- Google Cloud VM
- GitHub version control
- PostgreSQL (future expansion)

### Design Principles
- Automation First. Intelligence Second. Autonomy Last.
- Privacy by Design.
- Vendor-Agnostic Integration.
- Deterministic Core with AI-Ready Architecture.
- Operator Simplicity Above All.
